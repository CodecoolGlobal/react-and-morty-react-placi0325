<!-- GETTING STARTED -->
## Getting Started

Please see the core dependencies and installation steps below:

### Prerequisites

<ul>
  <li>Node.js</li>
</ul>

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/CodecoolGlobal/react-and-morty-react-placi0325
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter in a CLI
   ```sh
   npm run start
   ```
